# Avatarium React Native Example

A working Expo app with a live 3D AI avatar. Run it on your phone in 2 minutes.

## Try with your own avatar

[Create your avatar →](https://dashboard.avatarium.ai)

Replace the `avatarId` in the code with your own from [dashboard.avatarium.ai](https://dashboard.avatarium.ai).

## Requirements
- [Expo Go](https://expo.dev/go) on your iOS or Android device

## Quick Start

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go — the avatar loads on your device immediately.

## Try your own avatar
Enter your avatar ID (from [dashboard.avatarium.ai](https://dashboard.avatarium.ai)) in the app to enable full voice + AI.
