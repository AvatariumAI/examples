# Avatarium Vue Example

A working demo of Avatarium in a Vue 3 + Vite app.

## Try with your own avatar

[Create your avatar →](https://dashboard.avatarium.ai)

Replace the `avatarId` in the code with your own from [dashboard.avatarium.ai](https://dashboard.avatarium.ai).

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 — you'll see a live avatar immediately.

## Try your own avatar

1. Sign up at [dashboard.avatarium.ai](https://dashboard.avatarium.ai)
2. Create an avatar and copy its ID
3. Enter your avatar ID in the app to unlock full voice + AI

## Install in your project

```bash
npm install @avatarium/js@beta
```

This example uses an `<iframe>` embed (the simplest approach). For full SDK control (speak, stop, volume), use `@avatarium/js`.
